Инструкция по работе с дашбордами

Что это

В папке dashboard находятся два HTML-дашборда:

plan_fact_dashboard.html — анализ выполнения плана продаж;
unit_economics_dashboard.html — анализ юнит-экономики и эффективности SKU.

Дашборды полностью автономны: для работы не требуется интернет, сервер или дополнительное ПО. Достаточно открыть файл в браузере.

Как открыть
Откройте папку dashboard.
Дважды кликните по нужному .html файлу.
Дашборд откроется в браузере.

При необходимости файл можно просто отправить коллеге — он откроется аналогичным образом.

Дашборд «План / Факт»

Позволяет оценить выполнение плана продаж.

В отчете доступны:

основные KPI (план, факт, отклонение и процент выполнения);
динамика выполнения плана по месяцам;
сводная таблица по брендам;
тепловая карта, позволяющая быстро определить периоды с отклонениями.

Для всех показателей используется цветовая индикация:

🟢 план выполнен;
🟡 есть отклонения;
🟠 требуется внимание;
🔴 критическое отклонение.

Дашборд «Юнит-экономика»

Используется для оценки эффективности ассортимента.

Содержит:

ключевые показатели (выручка, прибыль, маржинальность, количество продаж);
анализ категорий и брендов;
ABC-анализ ассортимента;
ТОП прибыльных и проблемных SKU;
поиск и фильтрацию по всему ассортименту.
Поиск и фильтрация

В нижней части дашборда можно:

найти товар по артикулу;
отфильтровать данные по бренду;
выбрать категорию;
отобразить товары с нужным уровнем маржинальности.

Все фильтры работают совместно, поэтому можно быстро найти нужную группу товаров по нескольким условиям одновременно.

Таблица автоматически пересчитывает количество найденных SKU, суммарную выручку, прибыль и среднюю маржинальность.

Условное форматирование

Во всех дашбордах используется единая логика отображения показателей:

🟢 показатель в пределах нормы;
🟡 требует внимания;
🟠 есть существенные отклонения;
🔴 критическое значение.
Если возникли проблемы

Если дашборд не открылся:

убедитесь, что открываете именно .html файл;
используйте любой современный браузер (Chrome, Edge и др.);
не открывайте файл через Excel.

Все данные в проекте являются демонстрационными и используются исключительно для презентации аналитического решения.
